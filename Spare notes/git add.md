**git add** - Add untracked files to the "**staging area**", where they are ready to be committed in one go. Add can also be used to submit modified files for a snapshot.

	$ git add index.html
	$ git status
	On branch master

	No commits yet

	Changes to be committed:
	  (use "git rm --cached <file>..." to unstage)
			new file:   index.html
